PyWPS-4 demo app

This is a demo PyWPS server, point your WPS client to {{ url }}. It exposes a few processes:

  • say_hello - returns a literal string output based on the input.
  • feature_count - counts number of features in the uploaded GML.
  • centroids - returns GeoJSON with centroids of features in the uploaded GML.

PyWPS documentation is at pywps.rtfd.org; source code and issue tracking at github.com/jachym/pywps-4. Code for this app is at github.com/jachym/pywps-4/tree/master/demo.
